Under the Hood

Technical Stack & Motion Design

The WISAL website is built on a modern front-end stack that balances blazing-fast performance with smooth, high-fidelity interactions. Designed with a custom-engineered animation framework, it provides an immersive, tactile browsing experience.

🚀 Core Technologies

The framework leverages Next.js 16 (App Router) and React 19 to provide solid server-rendered SEO-friendly structures, paired with client-side reactive rendering. Styling uses TailwindCSS v4, integrating modern CSS custom properties and fluid typography values directly into utility classes.

✨ WebGL Background Canvas (Three.js)

A background canvas is managed in the DOM using Three.js, spawning a particle simulation system. In Celebration Mode, 300 floating gold dust particles drift across a sinusoidal pattern. In Memorial Mode, the system scales down to 150 slow-moving stardust particles to establish a serene environment. The particles dynamically drift away from the cursor using mathematical distance calculations and accelerate on page scroll.

🌀 Scroll Physics (Lenis + GSAP ScrollTrigger)

To ensure motion feels premium and avoids the default jagged browser scroll, the system runs Lenis Scroll. This intercepts the viewport scrolling velocity, smoothing the inertia. Combined with GSAP (GreenSock Animation Platform) and ScrollTrigger, structural elements fade and slide into place sequentially as they enter the screen, mimicking high-end digital luxury catalogs.

Operational Strategy

Client Inquiry & Dispatch Flow

The website functions not just as a static portfolio, but as a live concierge dispatcher. The client acquisition pipeline has been optimized to ensure inquiries are routed immediately to coordinators based in Amman.

📥 Stateful Booking Pipeline

The consultation form uses a single state-hook object in React that dynamically tracks client selections (Event date, Guest count, expected venue, budget range, and custom notes). All form inputs are validated client-side to ensure required information (Full name, phone, and category) is filled out prior to dispatch.

💬 WhatsApp Concierge API Dispatch

When the user submits the form, instead of relying on slow, un-read emails, the system triggers a direct redirect using the https://wa.me/ endpoint. The React state object compiles all inputs into a beautiful, pre-written, and readable WhatsApp message. For example: "Hello WISAL, I would like to request a consultation. Name: Ahmad. Phone: 079... Event Type: Wedding on 2026-09-12 at The Ritz-Carlton Amman...". This provides an instant and high-touch VIP response rate.

🌍 W3C Multilingual Standards

To accommodate both local Jordanian families and international corporate clients, the site enforces complete language separation. Language changes dynamically toggle the browser root attributes (document.documentElement.lang and document.documentElement.dir) to shift between Left-to-Right (LTR) and Right-to-Left (RTL) mode. Fonts, flexbox alignments, text paddings, and GSAP slide vectors adjust instantly to preserve perfect editorial layouts.
